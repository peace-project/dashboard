About the PEACE Dashboard

The PEAcE dashboard is a collaborative effort between the Distributed Systems Group at University of Bamberg, the Institute of Architecture of Application Systems at University of Stuttgart, and the Faculty of Informatics at Università della Svizerra italiana to present all the data of workflow engines generated by the betsy and the benchflow project.

The dashboard itself has been created as part of a student project at the Distributed Systems Group at University of Bamberg. It is built as a simple HTML5 application working on multiple JSON files which can be statically hosted. The data output of either betsy or benchflow can be automatically added to these JSON files through an additional transformation step.
